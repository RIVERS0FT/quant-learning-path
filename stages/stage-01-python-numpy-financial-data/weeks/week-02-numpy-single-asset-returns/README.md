## 核心公式

简单收益率：

$$
R_t=\frac{P_t}{P_{t-1}}-1
$$

对数收益率：

$$
r_t=\ln\left(\frac{P_t}{P_{t-1}}\right)
$$

累计净值：

$$
NAV_t=\prod_{k=1}^{t}(1+R_k)
$$
