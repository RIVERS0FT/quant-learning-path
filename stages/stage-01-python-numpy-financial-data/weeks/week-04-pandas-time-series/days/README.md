# 第 4 周每日学习路径

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | Series | 索引、值、缺失值与向量化运算 | 单股票收盘价 Series |
| 2 | DataFrame | 列选择、行选择、布尔筛选 | 日线行情 DataFrame |
| 3 | 日期索引 | `to_datetime`、排序、切片、重采样概念 | 标准日期索引 |
| 4 | 文件读写 | CSV 编码、类型指定、Parquet | 两种格式行情文件 |
| 5 | 长表与宽表 | `pivot`、`melt`、多股票结构转换 | 长宽表转换脚本 |
| 6 | 字段规范 | 代码、日期、OHLCV、复权字段规则 | 行情字段字典 |
| 7 | 综合整理 | 清洗、排序、去重、保存与验证 | 标准行情表 |

## 标准字段建议

`trade_date`、`symbol`、`open`、`high`、`low`、`close`、`volume`、`amount`。

## 检查清单

- [ ] 日期字段为日期类型
- [ ] 股票代码保留前导零
- [ ] 主键无重复
- [ ] 文件重新读取后类型一致
