# 第 1 周每日学习路径

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | Python 环境 | 安装 Python、创建虚拟环境、配置编辑器和解释器 | 可运行的项目环境 |
| 2 | 变量与数据类型 | 数值、字符串、布尔值、列表、元组、字典、类型转换 | 基础类型练习脚本 |
| 3 | 条件与循环 | `if`、`for`、`while`、`range`、`break`、`continue` | 简单价格遍历程序 |
| 4 | 函数 | 参数、返回值、默认参数、类型提示、文档字符串 | 简单收益率函数 |
| 5 | 异常与路径 | `try/except`、`pathlib`、文件读写 | 安全读取价格文件 |
| 6 | Git 基础 | `init`、`status`、`add`、`commit`、`.gitignore` | 首次规范提交 |
| 7 | 综合项目 | 整理目录、补充 README、测试收益率函数 | 第 1 周项目提交 |
