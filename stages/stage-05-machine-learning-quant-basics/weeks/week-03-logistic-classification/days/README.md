# 第 3 周：逻辑回归与分类

## 周目标

将选股问题转为上涨、超额或分位分类并评价概率输出。

## 核心知识

- 分类标签
- 逻辑函数
- 概率阈值
- 校准评价

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | 分类标签 | 理解定义、目标、假设和适用边界 | 分类标签笔记 |
| 2 | 逻辑函数 | 整理数据、接口、时间语义和约束 | 逻辑函数规范 |
| 3 | 概率阈值 | 实现最小可验证基准并构造样例 | 概率阈值基准 |
| 4 | 校准评价 | 加入扩展功能并处理异常和边界 | 校准评价模块 |
| 5 | 系统集成 | 接入已有系统并核验“逻辑回归与分类”输入输出 | 逻辑回归与分类集成样例 |
| 6 | 测试审查 | 执行测试、风险检查或故障演练，记录失败案例 | 逻辑回归与分类审查报告 |
| 7 | 周项目 | 整理代码、配置和文档，交付“分类基准与概率报告” | 分类基准与概率报告 |

## 周成果

分类基准与概率报告。
