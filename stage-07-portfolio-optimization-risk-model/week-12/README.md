# 第 12 周：优化器工程实现与组合优化阶段项目

## 本周目标

- 理解优化器工程实现的核心概念、适用条件、工程接口、验证方法与主要风险。
- 整合协方差估计、风险模型、约束优化、交易成本和风险归因。
- 完成带约束和成本的组合优化模块，并形成可复现阶段报告。

## 核心成果

交付组合优化模块、自动测试、风险与成本报告以及阶段总结。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

## 完成标准

- 能脱离代码解释优化目标、约束、假设和不适用场景。
- 输入、输出、数据时点、配置、版本和约束均有记录。
- 协方差、风险模型、优化器、成本和归因形成完整链路。
- 至少覆盖正常、边界、不可行和数值失败案例。
- 新环境可依据 README 复现阶段项目。

## 每日学习路径

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | 优化目标与边界 | 固化目标函数、约束、适用条件和失败方式 | 概念笔记与问题清单 |
