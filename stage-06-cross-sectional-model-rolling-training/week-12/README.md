# 第 12 周：成本感知训练、评估与截面选股阶段项目

## 本周目标

- 理解成本感知训练与评估的核心概念、适用条件、工程接口、验证方法与主要风险。
- 整合截面数据集、标签、滚动训练、模型更新、调仓和成本约束。
