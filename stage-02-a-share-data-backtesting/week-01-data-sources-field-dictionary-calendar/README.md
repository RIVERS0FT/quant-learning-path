# 第 1 周：数据源、字段字典与交易日历

## 本周目标

- 比较不同 A 股日线数据源的覆盖范围、更新时间和授权限制。
- 建立统一字段字典、主键和数据类型规范。
- 生成可复用的交易日历并处理非交易日。

## 核心成果

形成数据源评估表、标准字段字典和交易日历模块。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

## 完成标准

- 字段名称、单位和价格口径明确。
- 交易日历覆盖研究区间且可自动更新。
- 原始数据与标准数据可以追溯。
