# 第 2 周：股票代码、证券状态与日期标准化

## 本周目标

- 统一交易所、证券代码和内部资产标识。
- 处理上市、退市、暂停上市、风险警示等历史状态。
- 统一交易日、公告日和数据入库时间的日期语义。

## 核心成果

建立证券主表和历史证券状态表。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

## 完成标准

- 同一证券具有稳定的内部标识。
- 状态变化按生效日期保存。
- 日期字段不存在字符串与时区混用。
