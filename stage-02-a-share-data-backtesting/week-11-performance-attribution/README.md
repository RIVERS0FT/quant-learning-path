# 第 11 周：绩效分析与归因

## 本周目标

- 建立策略、基准和超额收益的统一评价。
- 分解持仓、交易、成本和行业暴露对结果的贡献。
- 生成可复现的回测报告。

## 核心成果

完成绩效分析模块和自动化报告。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

## 完成标准

- 收益、风险和回撤指标口径一致。
- 交易成本前后结果可以对照。
- 指标可从每日账户与交易明细重新计算。

## 每日学习路径

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | 净值与收益 | 从账户快照生成策略净值、日收益和累计收益 | 收益序列 |
| 2 | 风险指标 | 计算波动率、最大回撤、夏普和 Calmar | 风险指标表 |
| 3 | 交易统计 | 统计换手率、持仓期、胜率、盈亏比和成交率 | 交易统计表 |
| 4 | 基准与超额 | 对齐基准交易日并计算超额与跟踪指标 | 相对绩效表 |
| 5 | 成本归因 | 分解佣金、税费、滑点和冲击成本 | 成本归因表 |
| 6 | 暴露分析 | 按行业、规模或自定义分组分析持仓贡献 | 暴露与贡献表 |
| 7 | 周项目 | 生成包含图表、指标和交易明细的 HTML 或 Markdown 报告 | 自动回测报告 |
