# 第 12 周：参数敏感性与稳健性测试

## 本周目标

- 识别参数调优、样本选择和数据处理带来的过拟合。
- 开展时间分段、参数网格和成本压力测试。
- 建立策略通过或淘汰的最低证据标准。

## 核心成果

完成稳健性测试框架和策略审查报告。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

## 完成标准

- 测试区间与参数选择在运行前确定。
- 结果不只展示最优参数。
- 失败实验和异常结果同样被记录。
