# 第 9 周：手续费、滑点与冲击成本

## 本周目标

- 建立佣金、税费和其他交易费用模型。
- 比较固定滑点、比例滑点和基于成交量的冲击模型。
- 分析成本对换手率和策略容量的影响。

## 核心成果

完成可配置交易成本模块和成本敏感性报告。
