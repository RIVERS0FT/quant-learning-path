# 第 3 周：日线清洗与异常检测

## 本周目标

- 建立 OHLCV 日线数据清洗流程。
- 识别重复、缺失、价格关系错误和异常跳变。
- 区分真实市场事件、停牌缺失与数据错误。

## 核心成果

完成日线清洗管道和数据质量报告。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

## 完成标准

- 清洗规则可配置且保留原始值。
- 异常记录包含股票、日期、规则和严重级别。
- 清洗结果通过自动测试。

## 每日学习路径

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | 完整性检查 | 检查主键重复、日期缺口和字段缺失 | 完整性报告 |
| 2 | OHLC 关系 | 验证最高价、最低价、开盘价和收盘价的业务关系 | 价格关系检查 |
| 3 | 成交字段 | 检查成交量、成交额、零成交和负值 | 成交异常清单 |
| 4 | 异常收益 | 使用分位数、稳健标准分和事件日信息识别跳变 | 异常跳变清单 |
| 5 | 停牌与缺失 | 区分无交易、停牌、缺源和结构性空值 | 缺失分类字段 |
| 6 | 清洗管道 | 组合规则、日志、隔离区和修复策略 | 清洗模块 |
| 7 | 周项目 | 对多股票样本生成质量评分与问题明细 | 数据质量报告 |
