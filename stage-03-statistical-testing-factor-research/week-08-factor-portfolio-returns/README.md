# 第 8 周：因子分组与多空组合

## 本周目标

通过分位数组合、单调性和多空收益评价因子的可交易表现。

## 核心成果

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | 分组方法 | 实现等数量、等市值和分位数组合 | 分组函数 |
| 2 | 组合收益 | 计算等权、市值权重和基准调整收益 | 组合收益表 |
