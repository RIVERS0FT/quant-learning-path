# 第 6 周：SQL、DuckDB 与数据存储

## 本周目标

- 掌握 SQL 查询、筛选、聚合、连接和窗口函数。
- 使用 DuckDB 查询本地 Parquet。
- 理解分区和查询效率。

## 核心成果

建立本地行情数据库并完成基础查询脚本。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

## 完成标准

- 能通过 SQL 查询指定股票和日期。
- 能完成多表连接和滚动统计。
- 数据文件和查询脚本分离。

## 每日学习路径

| 天 | 主题 | 学习与实践 | 当天输出 |
|---:|---|---|---|
| 1 | SQL 基础 | `SELECT`、`WHERE`、`ORDER BY`、`LIMIT` | 基础查询脚本 |
| 2 | 聚合 | `GROUP BY`、`COUNT`、`AVG`、`STDDEV` | 股票统计查询 |
| 3 | 连接 | `INNER JOIN`、`LEFT JOIN`、主键设计 | 行情与股票信息连接 |
| 4 | 窗口函数 | `LAG`、`ROW_NUMBER`、移动窗口 | SQL 收益率计算 |
| 5 | DuckDB | 直接查询 CSV 与 Parquet | DuckDB 查询脚本 |
| 6 | 分区与效率 | 按年份、月份或市场分区 | 分区数据目录 |
| 7 | 综合项目 | 建库、查询、性能对比和文档 | 本地行情数据库 |
