# 阶段 01：Python、NumPy 与金融数据基础

**时间：第 1—3 个月**

## 阶段目标

- 掌握 Python、NumPy、pandas、SQL 与 DuckDB 基础。
- 能够处理 A 股日线数据并计算基础金融指标。
- 建立可复用的收益率、统计分析和数据质量模块。
