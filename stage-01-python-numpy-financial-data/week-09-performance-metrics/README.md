# 第 9 周：策略绩效指标

## 本周目标

- 掌握年化收益、年化波动率、夏普比率和最大回撤。
- 理解 Calmar、胜率、盈亏比与超额收益。
- 建立统一的绩效评价接口。

## 核心成果

编写统一绩效评价模块。

## 每日安排

每日学习内容与索引均直接保存在当前周目录。

- 模块包含测试和示例。
