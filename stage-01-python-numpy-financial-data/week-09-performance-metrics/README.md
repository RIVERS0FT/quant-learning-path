## 核心公式

夏普比率：

\[
Sharpe
=
\frac{R_p-R_f}{\sigma_p}
\]

最大回撤：

\[
MDD
=
\min_t\left(
\frac{V_t-\max_{s\le t}V_s}{\max_{s\le t}V_s}
\right)
\]
