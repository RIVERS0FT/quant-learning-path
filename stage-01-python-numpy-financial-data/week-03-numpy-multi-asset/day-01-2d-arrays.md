## 一、从单资产升级到多资产

单资产价格通常写成一维数组：

```python
prices = np.array([10.0, 10.2, 10.1, 10.5])
```

多资产价格使用二维矩阵：

```python
import numpy as np

prices = np.array(
    [
        [10.00, 20.00, 30.00],
        [10.20, 19.80, 30.60],
        [10.10, 20.40, 30.30],
        [10.50, 20.80, 31.20],
        [10.80, 20.50, 31.80],
    ],
    dtype=np.float64,
)
```

约定：

```text
行：交易日
列：股票
```

对应表格：

| 日期 | 股票 A | 股票 B | 股票 C |
|---|---:|---:|---:|
| 第 1 日 | 10.00 | 20.00 | 30.00 |
| 第 2 日 | 10.20 | 19.80 | 30.60 |
| 第 3 日 | 10.10 | 20.40 | 30.30 |
| 第 4 日 | 10.50 | 20.80 | 31.20 |
| 第 5 日 | 10.80 | 20.50 | 31.80 |

## 二、二维数组属性

```python
print(prices.ndim)
print(prices.shape)
print(prices.size)
print(prices.dtype)
```

结果：

```text
维度：2
形状：(5, 3)
元素数量：15
数据类型：float64
```

可以拆分形状：

```python
n_days, n_assets = prices.shape
```

其中：

- `n_days == 5`
- `n_assets == 3`

## 三、获取单个元素

二维索引格式：

```python
prices[行索引, 列索引]
```

示例：

```python
prices[0, 0]    # 第 1 日股票 A
prices[1, 2]    # 第 2 日股票 C
prices[-1, -1]  # 最后一天最后一只股票
```

---

## 四、获取一行：某天全部股票

```python
first_day_prices = prices[0, :]
last_day_prices = prices[-1, :]
```

也可简写：

```python
first_day_prices = prices[0]
```

一行表示某个时间点所有股票的数据，这类数据称为**横截面数据**。

---

## 五、获取一列：某只股票完整时间序列

```python
stock_a = prices[:, 0]
stock_b = prices[:, 1]
stock_c = prices[:, 2]
```

一列表示一只股票随时间变化的价格序列，这类数据称为**时间序列数据**。

## 六、二维切片

获取前三个交易日：

```python
prices[:3, :]
```

获取前两只股票：

```python
prices[:, :2]
```

获取第 2 日至第 4 日的股票 B、C：

```python
subset = prices[1:4, 1:3]
```

切片仍然遵循左闭右开规则。

## 七、多资产收益率矩阵

前一日价格矩阵：

```python
previous_prices = prices[:-1, :]
```

当前价格矩阵：

```python
current_prices = prices[1:, :]
```

两个矩阵形状都是：

```text
(4, 3)
```

简单收益率：

$$
R_{t,i}=\frac{P_{t,i}}{P_{t-1,i}}-1
$$

NumPy 实现：

```python
returns = current_prices / previous_prices - 1
```

结果约为：

```text
[[ 0.02000000 -0.01000000  0.02000000]
 [-0.00980392  0.03030303 -0.00980392]
 [ 0.03960396  0.01960784  0.02970297]
 [ 0.02857143 -0.01442308  0.01923077]]
```

价格矩阵形状为 `(5, 3)`，收益率矩阵形状为 `(4, 3)`。股票数量不变，时间维度减少 1。

## 八、逐元素运算与矩阵乘法

相同形状数组使用 `+`、`-`、`*`、`/` 时，NumPy 默认逐元素运算。

```python
a * b
```

不是线性代数中的矩阵乘法。

矩阵乘法使用：

```python
a @ b
```

多资产价格与收益率计算主要依赖逐元素运算。

## 九、初步理解 axis

按每只股票计算平均价格：

```python
mean_by_asset = prices.mean(axis=0)
```

结果形状为 `(3,)`，每只股票得到一个值。

按每天计算全部股票平均价格：

```python
mean_by_day = prices.mean(axis=1)
```

结果形状为 `(5,)`，每天得到一个值。

实用记忆：

- `axis=0`：压缩行，保留列，得到每只股票的结果；
- `axis=1`：压缩列，保留行，得到每天的结果。

---

## 十、完整示例

```python
import numpy as np


def main() -> None:
    prices = np.array(
        [
            [10.00, 20.00, 30.00],
            [10.20, 19.80, 30.60],
            [10.10, 20.40, 30.30],
            [10.50, 20.80, 31.20],
            [10.80, 20.50, 31.80],
        ],
        dtype=np.float64,
    )

    n_days, n_assets = prices.shape
    returns = prices[1:, :] / prices[:-1, :] - 1

    print("交易日数量：", n_days)
    print("股票数量：", n_assets)
    print("第一天全部股票：", prices[0])
    print("股票 A 时间序列：", prices[:, 0])
    print("收益率矩阵：")
    print(returns)
    print("每只股票平均价格：", prices.mean(axis=0))
    print("每日股票平均价格：", prices.mean(axis=1))


if __name__ == "__main__":
    main()
```
