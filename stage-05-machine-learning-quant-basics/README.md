# 阶段 05：机器学习量化基础

**时间：第 13—15 个月**

## 阶段目标

掌握面向截面选股的机器学习基本流程，重点理解数据泄漏、时间验证、基准模型、解释性与稳健性。

## 学习导航

各周学习计划见本目录下的 `week-*` 子目录。

## 阶段项目

完成一个使用线性模型、树模型和提升模型预测未来截面收益或排名的机器学习基准项目。

## 完成标准

- 12 周均具有明确周目标、7 天学习路径和验收清单。
- 核心模块具备接口、配置、日志、测试和复现说明。
- 数据、代码、参数、模型或策略版本可追踪。
- 结果包含效果、风险、成本、稳定性和失败案例。
- 完成阶段项目、总结报告与下一阶段计划。
