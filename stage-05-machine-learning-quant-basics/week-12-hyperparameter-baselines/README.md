## 完成标准

- 搜索空间、次数、随机种子和验证窗口可追踪。
- 测试集不参与模型选择。
- 复杂度提升必须带来稳定样本外增益。
- 线性模型、树模型和提升模型使用统一口径比较。
- 模型效果、风险、成本、解释性和失败案例进入模型卡。
- 新环境可依据 README 复现主要实验。
